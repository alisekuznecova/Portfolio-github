Group project
Task: Each day choose a different given animations to code together with another student.

For the first day i chose to code simple calculator, which has 4 different functions-plus, minus, divide and multiply. The numbers have to be written as a text in the text boxes and then the wanted function(which is displayed as a button) has to be clicked in order to get the result. For second day i chose something different to try to use function, which creates random and unpredictable result- for this aim, the perfect choice was the “Smack the fly” game. To start the game user has to click on the fly-after succeeding at smacking the fly, he can restart the game by clicking on it one more time.
